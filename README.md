# 🌍 Tour & Travel Web App (Django REST Framework)

A full-stack **Tour and Travel Web Application** built using **Django and Django REST Framework (DRF)**. This project allows users to explore tour packages, view details, and manage travel-related data through RESTful APIs.

---

## 🚀 Features

* User authentication (Signup/Login)
* Browse and view tour packages
* RESTful APIs for data handling
* Admin panel for managing tours and users
* Efficient database management using Django ORM

---

## 🛠️ Tech Stack

* Python
* Django
* Django REST Framework (DRF)
* SQLite
* HTML, CSS

## ⚙️ Installation & Setup

```bash
git clone https://github.com/krishd-bot/Tour-Travel-Web-app.git
cd Tour-Travel-Web-app

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate
python manage.py runserver
```
